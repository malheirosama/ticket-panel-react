# Ticket Panel React

Painel simples para gerenciamento de chamados desenvolvido em **React + TypeScript**.

## Funcionalidades

- Criar novos chamados
- Alterar status (Aberto / Em andamento / Resolvido)
- Filtrar chamados por status
- Buscar chamados por cliente ou título
- Interface simples e responsiva

## Tecnologias utilizadas

- React
- TypeScript
- Vite
- Custom Hooks
- Component-based architecture
